# GL-003 Brand — Dart Buddies

**inherits from:** [[GL-003-brands/dartscoaching]]

> **Part of the multi-brand design system.** This file is the token SSOT for **Dart Buddies** only. Before reading further, confirm this is actually the brand the task is for — see [[GL-003-design-system]] §1 for the cold-start rule every agent follows. Never read this file on a hunch; the briefing must say "Dart Buddies" explicitly.

> **Dart Buddies is a sub-brand of DartsCoaching.nl, not an independent brand.** It inherits the full identity, color palette, typography, spacing scale, imagery style, and voice samples from [[GL-003-brands/dartscoaching]]. This file does **not** repeat those sections. It only lists what is explicitly different for Dart Buddies, in the Overrides block below.
>
> **Reading order for any agent working on Dart Buddies:**
> 1. Read [[GL-003-brands/dartscoaching]] in full — that is the base.
> 2. Read the Overrides block below.
> 3. Apply the overrides on top of the base. Every section not mentioned below is identical to DartsCoaching.nl — do not assume a difference exists just because this file is short.
>
> If a new Dart Buddies-specific divergence is identified later (a new color, a different type choice, anything), Harmonia adds it as a new line under Overrides here. It does not get duplicated into the base DartsCoaching.nl file, and it does not get invented ahead of time — see the "empty is honest, placeholder is dangerous" principle in [[GL-003-design-system]].

## Overrides

| Section | Override | Value |
|---|---|---|
| Imagery style | Extra figuren (naast de gedeelde basisset) | `<nog niet ingevuld>` |

*Correctie (2026-07-07): de kern-avatarset (7 coach-personages + 4 klantpersona's Nina/Bob/Weszley/Willie, zie [[GL-003-brands/dartscoaching]] §5) is **geen** Dart Buddies-specifieke override — die set is gedeeld en wordt door zowel DartsCoaching.nl als Dart Buddies gebruikt, dus hij staat in de basis (dartscoaching.md), niet hier. Wat Dart Buddies wél specifiek heeft, zijn **extra figuren bovenop die gedeelde set** — Sander noemde dit expliciet, maar de concrete figuren zijn nog niet vastgelegd.*

*Deze tabel groeit alleen wanneer Sander en Harmonia samen een Dart Buddies-specifieke waarde vastpinnen via [[SOP-006-author-a-design-system]]. Een leeg-ogende waarde betekent "nog niet besloten" — niet "hetzelfde als DartsCoaching.nl's default". Tot de extra figuren zijn vastgelegd, flaggen agents die Dart Buddies-beeldwerk bouwen "GL-003-brands/dartbuddies.md — extra figuren nog niet ingevuld" en werken ze met alleen de gedeelde basisset, of routeren ze naar Harmonia.*
