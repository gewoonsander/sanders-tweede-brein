# Leren vibecoden voor websites

## Why this matters

Sander heeft een klein WordPress-bedrijfje (Gewoon Sander) waarmee hij websites host voor klanten, waaronder RDB. Ook zijn eigen site (gewoonsander.nl) en de site van zijn vrouw (praktijk) draaien erop. Hij wil dit werk niet loslaten, maar ook niet er volledig in duiken.

Vibecoding maakt het mogelijk om zelfstandig te sleutelen aan websites zonder alles technisch te hoeven kennen. Sander begrijpt WordPress al — Claude kan de uitvoering doen. Dit sluit aan bij zijn brede wens om te blijven leren en nieuwe tools te verkennen.

Mogelijke richtingen:
- Eigen site aanpassen (gewoonsander.nl)
- Site van zijn vrouw onderhouden
- WordPress-plugins of interfaces bouwen met Claude Code / Cursor
- Mogelijk: klantwerk afstoten of juist uitbreiden met nieuwe vaardigheden

## Definition of done

Sander heeft minstens één keer zelfstandig een wijziging aan een eigen website gedaan via vibecoding (Claude Code of Cursor), zonder externe hulp.
