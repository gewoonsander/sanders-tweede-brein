## Shared style spec (paste into every prompt)

> Flat illustration avatar. Minimalist, modern app-style, inspired by Duolingo characters and Google Illustrations. Round face, simple geometric body shape, clean bold outlines, limited color palette (4–6 flat colors per character, no gradients). Friendly expression. Transparent background. 512×512px. No text. No shading beyond flat color fills. No photo-realistic textures. Avatar must read well as a circle crop in a dark UI.

Use this block verbatim at the end of every character prompt below.

## Recommended tools

| Tool | Notes |
|---|---|
| **Midjourney v6+** | Best flat illustration consistency. Use `--ar 1:1 --style raw` and append `--no gradients, no shadows` |
| **DALL-E 3** (ChatGPT) | Good at following detailed prompts. Use "flat illustration" and "512x512" explicitly. |
| **Adobe Firefly** | Strong on style consistency across a set. Use "flat vector illustration" style. |
| **Canva AI** | Quick and easy. Style: "Illustration > Flat". Upload first result as style reference for the rest. |
| **Ideogram** | Strong text/style consistency. Use style tag: `flat illustration`. |

## Iteration suggestions (if first pass misses)

1. Add "vector art, SVG-style, no texture, paper-flat" to any prompt that comes back too photographic.
2. If the character loses the mythological attribute, add it twice: once in the subject description, once in the negatives as "do not omit the [attribute]".
3. For style consistency across all 12: generate Hermes first. Save the output. Use it as a style reference image for all subsequent characters ("match the illustration style of this image").
